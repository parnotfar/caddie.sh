# Debug Module

The Debug module manages Caddie's global debug flag and debug output.

## Overview

Use this module to:
- Enable or disable debug logging
- Check whether debug is on

## Commands

- `caddie debug:on` - Enable debug output
- `caddie debug:off` - Disable debug output
- `caddie debug:status` - Show current debug status
- `caddie debug:help` - Show debug help

## Examples

```bash
caddie debug:on
caddie debug:status
caddie debug:off
```
